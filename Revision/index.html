🎯 Exam Intelligence — What sir tests

Common question types:

  • Scenario → identify the problem, name it, fix it
  • Construct confusion matrix from numbers
  • Calculate metric (Precision, Recall, F1, KS-D)
  • Compare two approaches (e.g. SMOTE vs Cost-Sensitive)
  • Draw/describe a correct pipeline (leakage questions)
  • Explain why X fails on Y geometry

Assignment 2 breakdown (exam mirror):

  • Q1: Confusion matrix, Accuracy vs Recall, SMOTE
  • Q2: K-Means failure, Linkage criteria, geometry
  • Q3: Data leakage, pipeline order
  • Q4: Drift types, KS Test, Red Teaming
