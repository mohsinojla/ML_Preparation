📍 W1 — KNN & Feature Scaling

Distance Metrics

Euclidean: $d=\sqrt{\sum(p_i-q_i)^2}$
Manhattan: $d=\sum|p_i-q_i|$
Minkowski: $d=\left(\sum|p_i-q_i|^r\right)^{1/r}$
  • KNN: assign majority vote of K nearest
  • Small K → overfit; Large K → underfit
  • Always scale before KNN

Feature Scaling

Min-Max: $x'=\frac{x-x_{min}}{x_{max}-x_{min}}\;\to[0,1]$
Z-Score: $x'=\frac{x-\mu}{\sigma}\;\to\mu=0,\sigma=1$

Gradient Descent

$\theta:=\theta-\alpha\frac{\partial J}{\partial\theta}$
  • Large $\alpha$ → diverges | Small $\alpha$ → slow
  • Update all $\theta$ simultaneously
📈 W2 — Regression & Regularization

Linear Regression

$h_\theta(\mathbf{x})=\boldsymbol{\theta}^T\mathbf{x}$
$J(\theta)=\frac{1}{2m}\sum(h_\theta(x^{(i)})-y^{(i)})^2$
Normal Eq: $\theta=(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$
  • MSE is convex → global min guaranteed
  • GD: $\frac{\partial J}{\partial\theta_0}=\frac{1}{m}\sum(h-y)$
  • $\frac{\partial J}{\partial\theta_1}=\frac{1}{m}\sum(h-y)\cdot x$

Bias-Variance

$\text{Error}=\text{Bias}^2+\text{Variance}+\sigma^2$
  • High bias = underfit (high train + test error)
  • High variance = overfit (low train, high test)
  • Irreducible noise $\sigma^2$ cannot be removed

Polynomial Regression

$h=\theta_0+\theta_1x+\theta_2x^2+\cdots+\theta_dx^d$
  • Still linear in $\theta$
  • Parsimony: use simplest model

Regularization

Ridge (L2): $J+\lambda\sum\theta_j^2$
Lasso (L1): $J+\lambda\sum|\theta_j|$
RidgeLasso
→ 0 exactlyNoYes ✅
Feature sel.NoYes ✅
Closed formYesNo
Trap: Never regularise $\theta_0$. $\lambda\to\infty$ → predict mean.
✂️ W3–W4 — Support Vector Machines

Hard Margin SVM

Boundary: $\mathbf{w}^T\mathbf{x}+b=0$
Margin: $\frac{2}{\|\mathbf{w}\|}$
$\min\frac{1}{2}\|\mathbf{w}\|^2$ s.t. $y_i(\mathbf{w}^T\mathbf{x}_i+b)\geq1$
  • Predict: $\hat{y}=\text{sign}(\mathbf{w}^T\mathbf{x}+b)$
  • Support vectors: $y_i(\mathbf{w}^T\mathbf{x}_i+b)=1$
  • Remove non-SV → same hyperplane

Soft Margin (C-SVM)

$\min\frac{1}{2}\|\mathbf{w}\|^2+C\sum\xi_i$
s.t. $y_i(\mathbf{w}^T\mathbf{x}_i+b)\geq1-\xi_i,\;\xi_i\geq0$
  • $\xi_i=0$: correct, outside margin
  • $0<\xi_i<1$: inside margin, correct side
  • $\xi_i\geq1$: misclassified
  • Large C → small margin, hard (→ overfit)
  • Small C → large margin, soft (→ underfit)

Kernel Trick

$K(\mathbf{x}_i,\mathbf{x}_j)=\langle\phi(\mathbf{x}_i),\phi(\mathbf{x}_j)\rangle$
Linear: $\mathbf{x}_i^T\mathbf{x}_j$
Poly: $(\mathbf{x}_i^T\mathbf{x}_j+c)^d$
RBF: $\exp(-\gamma\|\mathbf{x}_i-\mathbf{x}_j\|^2)$
  • RBF → ∞-dim space, $O(d)$ compute
  • Dual: replace $\langle\mathbf{x}_i,\mathbf{x}_j\rangle$ with $K(\mathbf{x}_i,\mathbf{x}_j)$
  • Dual: $0\leq\alpha_i\leq C$
🌳 W5 — Decision Trees

Impurity Measures

Entropy: $H(S)=-\sum_k p_k\log_2 p_k$
Info Gain: $\text{IG}(S,A)=H(S)-\sum_v\frac{|S_v|}{|S|}H(S_v)$
SplitInfo: $-\sum_v\frac{|S_v|}{|S|}\log_2\frac{|S_v|}{|S|}$
Gain Ratio: $\text{IG}/\text{SplitInfo}$
Gini: $1-\sum_k p_k^2$
  • Pure node: H=0 | Max H=1 (binary 50/50)
  • ID3 → max IG | C4.5 → Gain Ratio | CART → Gini

Pruning

Cost-Complexity: $R_\alpha(T)=R(T)+\alpha|T|$
Pre-PruningPost-Pruning
WhenDuring growthAfter full tree
RiskShort-sightedHindsight (better)
SpeedFastSlower
  • DT boundaries: always axis-parallel rectangles
  • Can represent any boolean function
  • Modern: use both (XGBoost: max_depth + gamma)
Trap: IG biased toward many-valued features → use Gain Ratio (C4.5).
⚙️ W6–W7 — Clustering

K-Means

Objective (Inertia): $J=\sum_k\sum_{i\in C_k}\|\mathbf{x}_i-\mu_k\|^2$
Assign: $c_i=\arg\min_k\|\mathbf{x}_i-\mu_k\|^2$
Update: $\mu_k=\frac{1}{|C_k|}\sum_{i\in C_k}\mathbf{x}_i$
  • Converges to local optimum (non-convex J)
  • More K → always lower inertia (trivial at K=N)
  • Elbow method → heuristic for K
Fails on: concentric rings, elongated shapes, unequal cluster sizes.

K-Means++ & X-Means

K-Means++

$P(\mathbf{x}\text{ chosen})\propto d(\mathbf{x})^2$
  • Far points → higher probability of selection
  • Avoids bad local optima. Default in sklearn.

X-Means

BIC: $-2\ln\hat{L}+p\cdot\ln n$
  • Auto-selects K by splitting if BIC improves
  • No need to specify K upfront

Hierarchical Clustering

  • Each point = own cluster → merge two closest → repeat
  • Result: dendrogram. Cut at height → K clusters.
LinkageDistanceGeometry
Singlemin distElongated, rings ✅
Completemax distCompact, spherical
Averagemean distBalanced
Exam: For concentric rings → single-linkage. Complete-linkage fails because max inter-cluster distance between rings is large → sees them as one cluster.
🚨 W8 — Anomaly Detection

Isolation Forest

$s(x,n)=2^{-E[h(x)]/c(n)}$, $c(n)=2H(n-1)-\frac{2(n-1)}{n}$
  • $s\approx1$ → anomaly | $s\approx0.5$ → normal | $s\approx0$ → very normal
  • Anomalies isolated in fewer splits → short path length

Gaussian Model

$p(\mathbf{x})=\prod_j\frac{1}{\sqrt{2\pi\sigma_j^2}}\exp\!\left(-\frac{(x_j-\mu_j)^2}{2\sigma_j^2}\right)$
  • Flag anomaly if $p(\mathbf{x})<\epsilon$
  • Anomaly detection preferred when positives are rare (0–20 examples), future anomalies may look different
📊 W10 — Model Evaluation

Confusion Matrix

Pred +Pred −
Act +TPFN
Act −FPTN
Accuracy $=\frac{TP+TN}{TP+TN+FP+FN}$
Fails when: imbalanced classes (99% TN → 99% acc, 0% recall).

Precision, Recall, F1

Precision $=\frac{TP}{TP+FP}$ — "how exact"
Recall $=\frac{TP}{TP+FN}$ — "how complete"
F1 $=\frac{2PR}{P+R}=\frac{2TP}{2TP+FP+FN}$
  • F1 is harmonic mean — penalises if P or R is very low
  • Use F1 when classes are imbalanced

SMOTE & Cost-Sensitive

SMOTE: $\mathbf{x}_{new}=\mathbf{x}_i+\lambda(\mathbf{x}_{nn}-\mathbf{x}_i)$, $\lambda\in[0,1]$
  • Interpolates between minority neighbours in feature space
  • Cost-sensitive: modify loss function weight, not data

Data Leakage

Train-Test Contamination: Fit scaler/encoder on full data before split → test stats leak into training.
Target Leakage: Using features only available after the event (post-diagnosis data to predict diagnosis).
✅ Correct: Split → Fit transformers on TRAIN only → Apply to test

Encoding & Cyclical

  • Label enc → false ordinal (bad for nominal)
  • One-hot → no false order, high dim
  • Target enc → mean(target) per category, but leakage risk
Cyclical: $x_{\sin}=\sin(\frac{2\pi x}{T}),\;x_{\cos}=\cos(\frac{2\pi x}{T})$
  • T=24 for hours; fixes 23→0 discontinuity
🚀 W12 — MLOps & FastAPI

MLOps Lifecycle

Data Collection → Data Prep → Model Training → Model Evaluation → Model Packaging → Deploy & Serve → Monitor & Retrain ← feedback loop

Fairness Metrics

Demographic Parity: $P(\hat{Y}=1|A=0)=P(\hat{Y}=1|A=1)$
