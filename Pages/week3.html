Support Vector Machines — Theory & Hard Margin

One of the most elegant and powerful classifiers in ML. Understand the geometry, margin maximisation, and the mathematical formulation of the hard-margin SVM.

🔍

Introduction to SVMs

▼

From the lecture: Support Vector Machines are non-probabilistic binary linear classifiers. They find the optimal hyperplane that separates two classes with the maximum possible margin.

The key idea: among all possible lines/planes that separate two classes, SVM picks the one that is furthest from all data points — maximising the margin between classes.

Geometric Intuition

❓ The Problem

Given labelled points (+1 and -1), there are infinitely many hyperplanes that correctly separate them. Which one do we pick?

✅ SVM's Answer

Pick the hyperplane with the maximum margin — the one that is as far as possible from the nearest point on each side.

🔑 Why Maximum Margin?
A larger margin means the model is more robust — small perturbations in data are less likely to cause misclassification. It gives the best generalisation from training to test data. This is backed by statistical learning theory (VC dimension).

Applications (From Lecture)

📄 Text Classification

Spam detection, sentiment analysis, news categorisation

🚨 Outlier Detection

One-class SVM finds anomalies in data (fraud, fault detection)

👤 Face Detection

Classical face/object recognition in computer vision

📌 Key Properties (from lecture)
  • Non-probabilistic: outputs a class label, not a probability
  • Binary: designed for 2-class problems (extended via one-vs-one or one-vs-rest)
  • Only a subset of training points define the classifier — these are the support vectors
  • Basis functions & kernel trick extend it beyond linear classifiers
✂️

Decision Boundary & Hyperplane

▼

In 2D, the decision boundary is a line. In 3D it's a plane. In $n$ dimensions it's a hyperplane. The SVM finds the optimal hyperplane.

Hyperplane Equation

Hyperplane
Decision Boundary Equation
$$\mathbf{w}^T \mathbf{x} + b = 0$$

$\mathbf{w}$ = weight vector (normal to the hyperplane) | $b$ = bias term | $\mathbf{x}$ = input vector

$\mathbf{w}$ (weight vector)

Points perpendicular to the hyperplane — defines its orientation.

$b$ (bias)

Shifts the hyperplane away from the origin — defines its position.

$\|\mathbf{w}\|$

The magnitude of $\mathbf{w}$ — appears in the margin calculation.

In 2D

$w_1 x_1 + w_2 x_2 + b = 0$ is just the equation of a line $Ax + By + C = 0$.

The distance from any point $\mathbf{x}_0$ to this hyperplane is: $\frac{|\mathbf{w}^T\mathbf{x}_0 + b|}{\|\mathbf{w}\|}$

Classifier Decision Rule (From Lecture)

Classification Rule
Sign of the decision function determines class
$$\hat{y} = \text{sign}(\mathbf{w}^T\mathbf{x} + b)$$ $$\hat{y} = \begin{cases} +1 & \text{if } \mathbf{w}^T\mathbf{x} + b \geq 0 \\ -1 & \text{if } \mathbf{w}^T\mathbf{x} + b < 0 \end{cases}$$
💡 Geometric Meaning
The sign of $\mathbf{w}^T\mathbf{x} + b$ tells you which side of the hyperplane a point is on. Positive → Class +1. Negative → Class -1. The hyperplane itself is where the value equals zero.
↔️

Margin & Support Vectors

▼

The margin is the total width of the "street" between the two classes — the distance between the two margin hyperplanes (one touching each class). SVM maximises this margin.

SVM Margin Diagram

🔵 Class +1 | 🔴 Class -1 | Green line = Decision boundary | Dashed = margin boundaries

Margin Width Formula

We define the two margin hyperplanes (gutters) as:

Margin Hyperplanes
Positive and Negative Margin Boundaries
$$\mathbf{w}^T\mathbf{x} + b = +1 \quad \text{(positive margin boundary)}$$ $$\mathbf{w}^T\mathbf{x} + b = -1 \quad \text{(negative margin boundary)}$$
Width
Total Margin Width
$$\text{margin} = \frac{2}{\|\mathbf{w}\|}$$
1
Distance from origin to positive boundary: $\frac{1 - b}{\|\mathbf{w}\|}$
2
Distance from origin to negative boundary: $\frac{-1 - b}{\|\mathbf{w}\|}$
3
Total margin = distance between the two boundaries = $\frac{2}{\|\mathbf{w}\|}$
🔑 Maximising Margin
Maximising $\frac{2}{\|\mathbf{w}\|}$ is equivalent to minimising $\|\mathbf{w}\|$, which is equivalent to minimising $\frac{1}{2}\|\mathbf{w}\|^2$ (easier to differentiate).

Support Vectors

Support vectors are the training points that lie exactly on the margin boundaries (i.e., $\mathbf{w}^T\mathbf{x} + b = \pm 1$). They are the only points that define the SVM.

🔑 Why "Support" Vectors?
If you removed any non-support-vector training point, the optimal hyperplane wouldn't change. Remove a support vector and the hyperplane shifts. They "support" (define) the margin boundaries.

Typically few SVs

Even with thousands of training points, there are usually just a handful of support vectors. This makes SVM memory-efficient.

Prediction uses only SVs

$\hat{y} = \text{sign}\left(\sum_{i \in \text{SVs}} \alpha_i y_i \langle\mathbf{x}_i, \mathbf{x}\rangle + b\right)$

🎯

The Objective Function (Primal Problem)

▼

SVM is a constrained optimisation problem. We want to find $\mathbf{w}$ and $b$ that maximise the margin while ensuring all points are correctly classified.

Primal Problem
Hard-Margin SVM Optimisation
$$\min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2$$ $$\text{subject to: } y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1 \quad \forall i$$

Understanding the Constraint

Constraint
Correct Classification + Outside Margin
$$y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$$

For Class +1 ($y_i = +1$)

Constraint becomes: $\mathbf{w}^T\mathbf{x}_i + b \geq +1$

Point must be on or beyond the positive margin

For Class -1 ($y_i = -1$)

Constraint becomes: $\mathbf{w}^T\mathbf{x}_i + b \leq -1$

Point must be on or beyond the negative margin

🔑 Elegant Trick
By labelling classes as $y_i \in \{-1, +1\}$ (not 0/1), the single constraint $y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$ simultaneously handles both classes. When $y_i = +1$ and prediction is positive, the product is $\geq 1$. When $y_i = -1$ and prediction is negative, the product is $\geq 1$ too.

Solving via Lagrange Multipliers

This constrained optimisation is solved using Lagrangian duality. We introduce Lagrange multipliers $\alpha_i \geq 0$ for each constraint:

Lagrangian
Primal Lagrangian
$$\mathcal{L}(\mathbf{w}, b, \boldsymbol{\alpha}) = \frac{1}{2}\|\mathbf{w}\|^2 - \sum_{i=1}^{m} \alpha_i \left[y_i(\mathbf{w}^T\mathbf{x}_i + b) - 1\right]$$
KKT Conditions
At the optimum, the KKT (Karush-Kuhn-Tucker) conditions give us: $\alpha_i = 0$ for non-support vectors (constraint is not active), and $\alpha_i > 0$ only for support vectors.

Week 3 Quiz

▼

🧩 Test Your Understanding

Q1. SVM is described as "non-probabilistic." What does this mean?

Non-probabilistic means SVM outputs a decision (class label) directly via the sign function, rather than computing class probabilities like logistic regression does.

Q2. The margin of an SVM is $\frac{2}{\|\mathbf{w}\|}$. Maximising the margin requires:

Since margin = 2/‖w‖, maximising this means minimising ‖w‖. We minimise ½‖w‖² instead because it's differentiable and equivalent.

Q3. Support vectors are:

Support vectors are the critical points that touch the margin hyperplanes. They are the only training points that matter for defining the optimal hyperplane.

Q4. For a point with $y_i = +1$, the hard-margin SVM constraint $y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$ becomes:

Substituting $y_i = +1$: $(+1)(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$, which simplifies directly to $\mathbf{w}^T\mathbf{x}_i + b \geq 1$.

Q5. What happens if you remove a non-support-vector training point from an SVM?

Only support vectors define the SVM. Non-support-vector points have Lagrange multiplier α_i = 0 and play no role in determining the hyperplane.

Numerical Problems

▼

Problem 1 — Compute Margin Width

An SVM has weight vector $\mathbf{w} = [3, 4]^T$. Compute the margin width.
Step 1
Compute $\|\mathbf{w}\| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$
Step 2
Margin width $= \frac{2}{\|\mathbf{w}\|} = \frac{2}{5} = \mathbf{0.4}$

Problem 2 — Classify a Point Using SVM

A trained SVM has $\mathbf{w} = [2, -1]^T$ and $b = -1$.
Classify the point $\mathbf{x} = [3, 1]^T$.
Step 1
Compute decision function: $\mathbf{w}^T\mathbf{x} + b = (2)(3) + (-1)(1) + (-1) = 6 - 1 - 1 = 4$
Step 2
Apply sign function: $\hat{y} = \text{sign}(4) = \mathbf{+1}$
Result
Point $(3, 1)$ is classified as Class +1 ✅
Distance from hyperplane: $\frac{|4|}{\|\mathbf{w}\|} = \frac{4}{\sqrt{5}} \approx 1.79$ (well within class +1 region)

Problem 3 — Verify Support Vectors

SVM with $\mathbf{w} = [1, 1]^T$, $b = -3$. Check which of these points are support vectors:
A = $(1, 2)$ with $y=+1$, B = $(3, 4)$ with $y=+1$, C = $(1, 0)$ with $y=-1$.
SV Condition
A point $\mathbf{x}_i$ is a support vector if $y_i(\mathbf{w}^T\mathbf{x}_i + b) = 1$
Point A
$\mathbf{w}^T\mathbf{x}_A + b = 1(1) + 1(2) - 3 = 0$
$y_A \times 0 = (+1)(0) = 0 \neq 1$ → NOT a support vector (on boundary!)
Point B
$\mathbf{w}^T\mathbf{x}_B + b = 1(3) + 1(4) - 3 = 4$
$y_B \times 4 = (+1)(4) = 4 \neq 1$ → NOT a support vector (well inside)
Point C
$\mathbf{w}^T\mathbf{x}_C + b = 1(1) + 1(0) - 3 = -2$
$y_C \times (-2) = (-1)(-2) = 2 \neq 1$ → NOT a support vector
Note
Support vectors satisfy the constraint with equality = 1. We'd need to find points exactly on the $\mathbf{w}^T\mathbf{x} + b = \pm 1$ hyperplanes. E.g., point $(2, 2)$ with $y=+1$: $1(2)+1(2)-3=1$ ✅ → Support Vector!
